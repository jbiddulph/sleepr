## Code Change Highlight

**Before (Problematic):**
```php
// This would fall back to anon key
$key = env('SUPABASE_SERVICE_ROLE_KEY') 
    ?? env('SUPABASE_SERVICE_KEY') 
    ?? env('SUPABASE_ANON_KEY');  // âš ï¸ Wrong key for uploads!
```

**After (Fixed):**
```php
// Only uses service role key
$key = env('SUPABASE_SERVICE_ROLE_KEY');

if (!$key) {
    // Clear error message instead of silent failure
    Log::error('SUPABASE_SERVICE_ROLE_KEY not set!');
    $this->status = 'Missing service role key';
    return;
}
```
